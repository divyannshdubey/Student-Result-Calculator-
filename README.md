<h1 align="center">🎓 Student Result Calculator</h1>

<p align="center">
A simple and interactive web-based application to calculate student results, percentage, and grade instantly.
</p>

<hr>

<h2>🚀 Features</h2>
<ul>
  <li>Enter student name and marks for 5 subjects</li>
  <li>Automatic total marks calculation</li>
  <li>Percentage calculation</li>
  <li>Grade assignment based on performance</li>
  <li>Clean and responsive UI design</li>
</ul>

<hr>

<h2>🛠️ Technologies Used</h2>
<ul>
  <li>HTML5</li>
  
</ul>

<hr>

<h2>📊 Grading System</h2>
<ul>
  <li>90% and above → A+</li>
  <li>75% – 89% → A</li>
  <li>60% – 74% → B</li>
  <li>50% – 59% → C</li>
  <li>Below 50% → Fail</li>
</ul>
